Web3产品

什么是Web3

Web3：以所有权为核心的价值网络

为了解决原始web网络无法支持普通人进行信息交流和分享的难题，一个“可读”的Web1时代伴随着万维网的发明而到来，各类搜索引擎和门户网站将信息大量搬上网，普通用户也能轻松访问各类资源。为了解决Web1下用户无法主动创建内容，不享有网络发言权的难题，一个“可读+可写”的Web2时代伴随着智能手机的普及、社交网络的兴起和云计算的发展而到来，所有用户都可以在社交媒体分享自己的所思所想并和他人交互，互联网世界的话语权从商业巨头转移到每一位终端用户手中。而现在，在经历了Web1和Web2时代后，互联网正在进行一场Web3革命。

Web3是用户拥有、用户控制的互联网。以太坊联合创始人加文·伍德（Gavin Wood）最初提出了Web3的概念，并将其描述为“一组保障人们在低壁垒市场中为自己行动”的兼容协议，认为其是“可执行的大宪章——个人反对暴力权威的自由的基础”；而知名区块链研究机构Messari的研究员江下（Eshita）及区块链技术公司BitFury CEO布里安·布鲁克斯（Brian Brooks），则分别将其定义为“可读+可写+拥有的互联网”、“用户可以拥有产权的互联网”。

人类信任模型及信任范式

社会元素、数学和物理规则、激励结构是构成人类网络信任模型的三个要件。

  • Web1时代是物理信任模型，由于开源的底层协议不受任何单一集体的控制，用户、开发者和企业都可以相信游戏规则将保持中立和公平。
  • Web2时代则是机构信任模型，由于协议是封闭的，商业公司接力数学和物理规则成为新的信任守门人，用户被迫信任其编写的专有和不透明的代码。
  • Web3的信任模型则是由数学和物理规则以及激励结构共同主导的可编程信任。开源协议以及区块链系统通过自动执行的智能合约、加密技术、共识机制、分布式技术等多种技术提供了可编程的信任；而代币发行和奖励则提供了基于激励结构的信任，用户无需再依赖信任中介。

Web3的革命性体现在哪

Web3以区块链为核心，解决价值分配问题和信任问题：区块链贯彻Web3去中心化的核心理念。通过建立共识协议、开放源代码，保证价值分配的可靠执行。作为安全性和去中心化水平都较高的网络可以通过共享账本储存数据、交换价值并记录交易活动，不受任何中心化实体控制，为Web3 提供安全的执行层，让使用者可以在其中创建、发行并交易加密资产，并且开发可编程的智能合约。

Web3 风险事件梳理

  • 2022年5月，UST-Luna崩盘事件：UST币是Terra公链上发行的算法稳定币，通过“LUNA-UST双币套利机制”维持币值稳定。UST发行方承诺，恒定1美元LUNA = 1个UST，当UST币的价格大于1美元，用户可以销毁价值1美元的LUNA币，来发行1枚UST币，反之亦然。Anchor的大规模提款和UST的大量抛售打破了表面稳定，将UST和Luna拖入下行螺旋。
  • 2022年6月，Celsius崩盘：2022年5月初，UST大幅下跌，大批资金撤离Celsius，储户挤兑加剧。Celsius的流动性管理无法满足兑付需求。当时Celsius持有的73%的ETH锁定在stETH或ETH2中，只有27%的ETH具备流动性。于是在2022年6月，Celsius宣布暂停提款，CEL代币一小时内大幅下跌70%。
  • 2022年11月，CoinDesk和CZ对于FTX业务稳定性的质疑引发了为期三天的约60 亿美元的挤兑，FTX 陷入流动性危机。随后，FTX申请破产保护，SBF被捕并面临多项欺诈指控。

Layer1，Layer2（以以太坊举例）

为了提升可扩展性，以太坊通过链上与链下两条路径积极进行扩容。

  • 链上扩容通常被称为Layer1，通过直接修改区块链规则来加快数据处理速度，扩充区块链容量。比如以太坊2.0通过将共识机制由PoW转向PoS、分片链等方式来解决可扩展性问题。

  • 而链下扩容则不改变区块链基础规则，转而通过架设外部通道，实现信息的传播与扩展，从而达到扩容效果。其中，Layer2解决方案的安全性直接建立在主网，主要包括状态通道、Plasma、Rollups、validiums等。其他解决方案包括各种形式的新链，其安全性独立于主网，如侧链等。

Layer2 RollUp技术方案

目前以太坊已确立起以Rollup为中心的扩容路线图，主要包括ZK Rollup和Optimistic两种类型。

1）交易速度：ZK Rollup理论上可以实现每秒数千笔的交易量，高于Optimism的每秒数百笔。

2）交易费用：ZK Rollup需要高端昂贵的硬件和证明验证，交易费用更高。

3）智能合约可执行性：ZK Rollup将智能合约逻辑转换为零知识证明，提高了智能合约的可执行性；Optimistic则需要更多的存储和计算资源来支持智能合约的执行。

4）安全性：ZK Rollup建立在零知识证明和密码学算法的基础上，可以保证交易隐私和链上安全；Optimistic则存在一定的安全风险。

5）去中心化程度：ZK Rollup需要更多的节点参与来确保去中心化程度。

钱包技术路径

多方计算、账户抽象和设备账户。 安全和易用性之间的平衡是钱包方案的发展关键。

  • 多方计算通过多个参与方共同计算，实现保护隐私数据的目的

  • 账户抽象将智能合约功能从特定账户中抽象出来，使智能合约可以被任何账户调用

  • 设备账户将设备作为账户，设备可以通过私钥签名的方式执行交易，并接收来自其他账户的交易。

海外广告岗位面试

概念

海外广告是指企业在本国以外的国家或地区进行的广告宣传活动。这种广告形式旨在向国际市场推广产品、服务或品牌，以扩大全球影响力和市场份额。以下是海外广告的主要特点和要素：

海外广告对于企业拓展国际市场、提高全球品牌认知度和实现收入多元化具有重要意义。成功的海外广告需要企业在全球视野和本地洞察之间找到平衡，并能够灵活应对不同市场的独特需求和挑战。

特点

  1. 跨文化传播：
    • 需要适应不同国家的文化背景
    • 考虑语言翻译和本地化
  2. 多样化渠道：
    • 传统媒体（电视、广播、平面）
    • 数字平台（社交媒体、搜索引擎、网站）
    • 户外广告（机场、公共交通）
  3. 法律法规遵从：
    • 符合目标国家的广告法规
    • 考虑不同地区的隐私保护政策
  4. 市场差异化：
    • 针对不同市场的消费者行为和偏好
    • 考虑经济发展水平和购买力差异
  5. 品牌一致性与本地化平衡：
    • 保持全球品牌形象一致性
    • 同时适应本地市场需求

主要类型

  1. 数字广告：
    • 社交媒体广告（Facebook, Instagram, Twitter）
    • 搜索引擎营销（Google Ads, Bing Ads）
    • 展示广告和程序化广告
    • 视频广告（YouTube, TikTok）
  2. 传统媒体广告：
    • 电视广告
    • 广播广告
    • 平面广告（报纸、杂志）
  3. 户外广告：
    • 广告牌
    • 交通广告（公交车、地铁）
    • 机场广告
  4. 内容营销：
    • 博客文章
    • 社交媒体内容
    • 视频内容
  5. 影响者营销：
    • 与当地知名人士合作
    • 社交媒体影响者合作
  6. 事件营销：
    • 赞助国际活动
    • 参加贸易展览会

挑战

  1. 文化差异：避免文化冲突和误解
  2. 语言障碍：准确传达品牌信息
  3. 法律合规：遵守不同国家的广告法规
  4. 市场研究：了解目标市场的消费者行为
  5. 预算分配：在多个市场间有效分配资源
  6. 效果衡量：跨国广告效果的准确评估

成功策略

  1. 深入的市场研究
  2. 本地化内容创作
  3. 选择适当的媒体渠道
  4. 与本地合作伙伴合作
  5. 灵活的广告策略
  6. 持续监测和优化
  7. 利用数据分析
  8. 建立全球-本地团队

现状分析与未来趋势

当前状况

  1. 数字化主导：
    • 社交媒体广告（Facebook, Instagram, LinkedIn）
    • 搜索引擎广告（Google Ads, Bing Ads）
    • 视频平台广告（YouTube, TikTok）
  2. 移动优先：
    移动设备使用率持续增长，移动广告成为主流
  3. 本地化策略：
    根据不同国家/地区的文化和消费习惯定制广告内容
  4. 数据驱动决策：
    利用大数据和AI进行精准定位和个性化推荐
  5. 跨平台整合：
    整合多个平台的广告策略，提供一致的品牌体验

未来趋势

  1. AI和机器学习的深度应用
    • 自动优化广告投放
    • 预测消费者行为
    • 个性化内容生成
  2. 增强现实（AR）和虚拟现实（VR）广告
    • 提供沉浸式广告体验
    • 虚拟试用产品
  3. 语音搜索优化
    • 适应智能音箱和语音助手的普及
    • 开发语音激活广告
  4. 视频内容的持续增长
    • 短视频广告（如TikTok）
    • 直播营销
  5. 隐私保护与无cookie广告
    • 适应GDPR等隐私法规
    • 开发不依赖第三方cookie的定位技术
  6. 可编程广告
    • 实时调整广告内容
    • 根据外部因素（如天气、新闻事件）动态变化
  7. 区块链在广告中的应用
    • 提高广告透明度
    • 减少广告欺诈
  8. 情境广告的兴起
    • 根据用户当前环境和情境投放相关广告
    • 利用IoT设备数据
  9. 社交商务整合
    • 社交媒体平台直接购物功能
    • 影响者营销的深度整合
  10. 可持续和道德广告
    • 强调品牌的社会责任
    • 环保和道德价值观的传播

挑战与机遇

挑战

  1. 数据隐私法规的收紧
  2. 广告屏蔽技术的普及
  3. 跨文化传播的复杂性
  4. 广告平台的快速变化
  5. 广告疲劳和用户注意力分散

机遇

  1. 新兴市场的增长潜力

  2. 技术创新带来的新广告形式

  3. 精准定位能力的提升

  4. 跨境电商的持续增长

  5. 内容营销与原生广告的融合

海外广告投放具体工作

1. 市场调研与策略制定

1.1 分析目标市场

  • 研究目标国家/地区的文化背景和消费习惯
  • 确定目标受众群体

1.2 竞品分析

  • 研究竞争对手的广告投放策略
  • 分析竞品优势，提炼可借鉴的经验

1.3 制定整体营销策略

  • 确定品牌定位和营销目标
  • 选择适合的广告平台（如Facebook、Google、TikTok等）
  • 制定预算分配方案

2. 广告创意与内容开发

2.1 设计广告文案

  • 根据目标市场特点创作吸引人的广告文案
  • 确保文案符合当地语言习惯和文化特点

2.2 制作广告创意

  • 设计视觉吸引力强的广告素材
  • 根据不同平台特点调整创意形式（如图片、视频、动画等）

2.3 优化产品展示页面（Listing）

  • 根据广告内容优化产品页面
  • 确保landing page与广告信息一致，提高转化率

3. 广告投放执行

3.1 选择广告投放渠道

  • 互联网平台（社交媒体、搜索引擎等）
  • 传统媒体（杂志、报纸）
  • 移动端应用
  • 户外流媒体

3.2 设置广告活动

  • 配置目标受众、预算、投放时间等参数
  • 上传广告创意和文案

3.3 实施竞价策略

  • 设置初始竞价
  • 根据平台特点调整竞价策略

4. 数据监控与分析

4.1 实时监控广告效果

  • 跟踪关键指标（如点击率、转化率、ROI等）
  • 设置预警机制，及时发现异常情况

4.2 收集和整理广告数据

  • 汇总各平台的广告表现数据
  • 整理用户行为和转化数据

4.3 建立数据分析模型

  • 构建搜索引擎市场营销模型
  • 开发自动化数据分析工具

5. 优化与调整

5.1 分析广告效果

  • 评估各渠道和广告组的表现
  • 识别表现优秀和欠佳的因素

5.2 优化关键词

  • 通过逻辑分析优化投放的关键词
  • 调整长尾关键词策略

5.3 改进广告投放策略

  • 调整预算分配
  • 优化目标受众定位
  • 更新广告创意和文案

5.4 产出优化报告

  • 总结优化策略和效果
  • 提供下一步优化建议

6. 持续改进与拓展

6.1 定期回顾和学习

  • 分析成功案例和失败教训
  • 跟踪行业最新趋势和技术

6.2 拓展海外市场资源

  • 寻找新的广告渠道和合作伙伴
  • 探索新兴市场机会

6.3 提升团队能力

  • 组织培训，提高团队的数据分析和创意能力
  • 鼓励创新，测试新的广告形式和策略

*专有名词

  1. CPC (Cost Per Click): 每次点击成本，广告主为每次广告点击支付的费用。

  2. CPM (Cost Per Mille): 千次展示成本，广告主为每千次广告展示支付的费用。

  3. CPA (Cost Per Action): 每次行动成本，广告主为每次用户完成特定行动（如注册、购买）支付的费用。

  4. CTR (Click-Through Rate): 点击率，广告被点击的次数与展示次数的比率。

  5. ROI (Return On Investment): 投资回报率，衡量广告投资的效益。

  6. KPI (Key Performance Indicator): 关键绩效指标，用于评估广告活动的成功程度。

  7. Conversion Rate: 转化率，完成目标行动的用户数量与总访问量的比率。

  8. Impressions: 展示次数，广告被显示的总次数。

  9. Reach: 触达人数，看到广告的独立用户数量。

  10. Frequency: 频次，平均每个用户看到广告的次数。

  11. Retargeting/Remarketing: 再营销，向之前与品牌互动过的用户投放广告。

  12. Lookalike Audience: 相似受众，与现有客户具有相似特征的新受众群体。

  13. A/B Testing: A/B测试，比较两个版本的广告以确定哪个效果更好。

  14. Landing Page: 着陆页，用户点击广告后到达的网页。

  15. Ad Rank: 广告排名，决定广告在搜索结果中的位置。

  16. Quality Score: 质量得分，衡量广告、关键词和着陆页的相关性和质量。

  17. PPC (Pay-Per-Click): 按点击付费，一种只在用户点击广告时才收费的广告模式。

  18. CTA (Call-To-Action): 行动号召，鼓励用户采取特定行动的文字或按钮。

  19. Ad Group: 广告组，包含一组相关广告和关键词的集合。

  20. Bid: 出价，广告主愿意为广告点击或展示支付的最高金额。

  21. Audience Targeting: 受众定位，根据用户特征选择广告受众的过程。

  22. Ad Extension: 广告扩展，为广告添加额外信息（如电话号码、地址）的功能。

  23. Negative Keywords: 否定关键词，防止广告在特定搜索查询中显示的关键词。

  24. ROAS (Return On Ad Spend): 广告支出回报率，衡量广告支出的直接收益。

  25. Funnel: 漏斗，描述用户从认知到转化的journey。
